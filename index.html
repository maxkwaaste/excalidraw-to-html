Why not just screenshot it?

Screenshots force Claude to guess. JSON gives it the facts.

📸 Screenshot approach

  • Claude interprets pixels -- approximates layout
  • Spacing and alignment are guesswork
  • Text in images can be misread
  • No info about connections between shapes
  • Small or overlapping elements get lost

{} JSON approach (this tool)

  • Exact element types: rectangle, diamond, text, arrow
  • Pixel-precise positions and dimensions
  • Text content read directly from data
  • Arrow bindings show which shapes connect
  • Nesting and grouping are explicit

What Claude actually reads

Every shape on your canvas is a JSON object with exact coordinates, types, and relationships.

// Claude reads structured data, not pixels rectangle: pos=(478,141) size=263x353 ← outer container rectangle: pos=(512,173) size=190x131 ← inner block 1 rectangle: pos=(513,341) size=190x131 ← inner block 2 arrow: pos=(739,306) size=180x1 ← connects container → diamond startBinding: "container" endBinding: "diamond" diamond: pos=(909,216) size=179x177 ← decision/output node

How shapes become HTML

Claude maps Excalidraw elements to semantic HTML based on type, position, and relationships.

You drawClaude generates
Large rectangle (contains others) Container, <section>, card wrapper
Small rectangle inside another Card, content block, nav item
Rectangle with text label Named section, button, heading
Diamond Decision point, CTA, status indicator
Ellipse Avatar, icon, badge
→ Arrow connecting shapes Flow relationship, navigation link
T Standalone text Heading, label, paragraph content

Setup in 5 minutes

Tell your Claude to clone this repo and follow the CLAUDE.md. Or do it manually:

1

Clone the MCP server

git clone https://github.com/yctimlin/mcp_excalidraw.git cd mcp_excalidraw && npm ci && npm run build
2

Register with Claude Code

claude mcp add excalidraw --scope user \ -e EXPRESS_SERVER_URL=http://localhost:3000 \ -e ENABLE_CANVAS_SYNC=true \ -- node $(pwd)/dist/index.js
3

Install the skill

Copy skill/SKILL.md from this repo to ~/.claude/skills/excalidraw-to-html/SKILL.md

4

Start the canvas

PORT=3000 npm run canvas

Opens the Excalidraw editor at http://localhost:3000

5

Draw and tell Claude

Draw your wireframe, then tell Claude: "Read my Excalidraw canvas and convert it to HTML"

Or just tell Claude to do it all

Paste this into Claude Code and it handles the entire setup for you.
